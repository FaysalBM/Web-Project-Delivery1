What Is Basical?

Basical Is a web application developed in Django that allows users to add a Company and they workmates along with them. This way, they're able to create departments inside to distribute the work. Also, inside every department there's Projects which are the one that conains the tasks that need to be done.

How can I use it?

On the Home screen you'll see a form to create the Company.

Important!

When you add the company, you'll be the only worker and admin on it. Then, alongside with the company name you'll see the a button to enter inside the Company and see the details.

Once you're inside the company detail page. You'll be able to add worker on It. This workers need to be users already registered on the website. After you add all the users you want on the comapny. With the form to add a department you'll create a new department inside the company

Once again, you'll be the obly user inside the department and the admin of this(Only you can delete this department). Same as the company, you'll see a button to access the detail of the department.

Finally, inside the department details, you'll see a list of the comapny workers. This way, you'll be able to add the workers you want inside that department. This way only some of the workers have access to that department.

Alongside with the department list to add users, there's a form to create a project inside the department. And inside the project there's another form to add a task, which will be attached to a form to modify the name of the task.
